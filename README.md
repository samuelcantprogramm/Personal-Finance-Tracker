# Personal Finance Tracker 

A beginner-friendly **Python + Tkinter (ttkbootstrap)** app to track your income, expenses, and savings. Ideal for students or anyone looking to manage finances easily.

---

## Features

- Add, edit, and delete income and expense entries
- Manual categories (food, bills, transport, etc.)
- Summary of total income, expenses, and balance
- Progress indicators per category
- Visual charts for expenses and savings
- Filter transactions by month
- Search transactions
- Data saved in CSV and exportable
